Liste des expressions valides pour programmer vos codebots.

Jouer

Instructions disponibles

Variables

var x = <expression>; → Déclaration d'une variable.

var x = <expression>; → Réaffectation d'une variable.

Types
  • 5 → Integer.
  • "hello" → Chaîne de caractères.
  • true ou false → Booléen.
  • [1, 2, 3] → Tableau.
  • {"key": "value"} → Paires clé-valeur.
  • fn(a, b) { <expressions> } → Fonction.
Opérateurs

Priorité des opérateurs du plus prioritaire au moins prioritaire.

  • (<expression>)
  • / et *
  • + et -
  • < et >
  • == et !=

Liste des opérateurs.

  • 5 + 3 → Addition.
  • 10 - 2 → Soustraction.
  • 4 * 1 → Multiplication.
  • 9 / 7 → Division.
  • 5 == 5 → Egalité.
  • 3 != 4 → Non-égalité.
  • 10 < 20 → Infériorité.
  • 15 > 7 → Supériorité.
  • !true → Négation.

Logique
  • if (<condition>) { <consequence> } else { <alternative> } → Condition.
  • while (<condition>) { <expressions> } → Boucle.
Fonctions
  • var add = fn(a, b) { return a + b; }; → Définir une fonction.
  • add(2, 3) → Appel de fonction.
Tableaux
  • var arr = [1, 2, 3, 4]; → Déclaration d'un tableau.
  • arr[1 + 1] → Accès à un élément (2).
Paires clé-valeur
  • var pierre = {"nom": "Pierre", "pv": 100}; → Déclaration d'une hash map.
  • pierre["nom"]; → Retourne "Pierre".
Fonctions existantes

Fonctions algorithmiques.

  • len("abc") → Retourne 3.
  • len([1,2,3]) → Retourne 3.
  • first([1,2,3]) → Retourne 1.
  • last([1,2,3]) → Retourne 3.
  • rest([1,2,3]) → Retourne [2,3].
  • push([1,2], 3) → Retourne [1,2,3].

Fonctions de contrôle du Codebot.

  • goto(<position>) → Déplace le robot jusqu'à la position indiquée.
  • canCarry(<itemType>) → Vérifie si le robot peut encore transporter un objet du type indiqué.
  • isEmpty() → Retourne vrai si l'inventaire du robot est vide.
  • wait(<time>) → Attend <time> ms.
  • has(<item>) → Retourne vrai si le robot possède l'objet (type et quantité).
  • hold(<itemType>) → Met l'objet du type indiqué dans la main du robot.
  • find(<resource>) → Retourne la position de la ressource la plus proche.
  • gather() → Récolte/miner la ressource présente sous le robot jusqu’à ce qu’elle disparaisse.
  • print(...<args>) → Affiche les arguments pendant un court instant.
  • craft(<item>) → Fabrique l'objet indiqué.
  • deposit(<item>) → Dépose l'objet indiqué dans un coffre.
  • take(<itemType>) → Prend l'objet indiqué depuis un coffre.
  • smelt(<item>) → Utilise le four pour faire fondre l'objet.

Types du Codebot.

  • <position> → {"x": <integer>, "y": <integer>}
  • <resource> →
  • <itemType> →
  • <item> → {"type": <itemType>, "amount": <integer>}

Exemple de programme

Répéter 3 fois : récolter et stocker du bois

var i = 0;
while (i < 3) {
    var pos = find("wood");
    goto(pos);
    gather();
    var i = i + 1;
}
